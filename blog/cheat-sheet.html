Javascript Objects

Javascript is a cross-platform, object-oriented scripting language invented in web browswers to make web pages more dynamic and give feedback to your user. Adding JavasScript to your HTML code allows you to change completely the document appearance, from changing text, to changing colors, or changing the options available in a drop-down list, or switching one image with another when you roll your mouse over it.

JS is mainly used as a client side scripting language. This means that all the action occurs on the client's side of things. When a user requests an HTML page with JS, the script is sent to the browser and it's up to the browswer to do something with it. JS is often used to perform operations that would otherwise encumber the server, like form input validation.

JS is supported by many popular web browswers including internet explorer and netscape navigator. One advantage of JS is an ability to get info about the browswer and HTML page. This access to info gives JS great power to modify the browsing experience.
